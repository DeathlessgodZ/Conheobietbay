Môn học: Cấu trúc dữ liệu và Giải thuật.

Mục tiêu của môn học là giúp sinh viên hiểu rõ về cách lưu trữ và sử lý dữ liệu.

Nội dụng bài giảng có thể được viết bằng chữ lớn để làm tiêu đề, hoặc chữ nhỏ để ghi chú.

Ví dụ công thức toán: n2 + 2n + 1 = (n+1)2.

Công thức hóa học bình thường dùng: H2SO4.

Kết quả khi chạy code có thể hiện thị trong thẻ smap.

Ví dụ code C++:
for (int i=0; i < 3; i++) {
cont << "data Structures" << endl;
}

Các tài liệu cũ được loại bỏ và thay bằng tài liệu mới.
